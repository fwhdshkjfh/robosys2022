# plusコマンド

* 標準入力から読み込んだ数字を足す。

## 実行例
```
$ seq 10 | ./plus
55
```


# 使用環境

* Ubuntu 18.04.5(動作確認済み)
* Python3.7~3.10(動作確認済み)
