# robosys2022
こちらは千葉工業大学先進工学部未来ロボティクス学科2年ロボットシステム学で取り扱っているリポジトリです。
___

# 本リポジトリの使用方法

```
$  git clone https://github.com/fwhdshkjfh/robosys2022.git

$ ls        #ダウンロードされたフォルダを確認

robosys2022

$ cd robosys2022
```
___

#  動作確認済み環境

* Python 3.7~3.10

* Ubuntu 18.04.5
